- **核心服务契约**：`mcp_server` 作为能力底座，通过 `skill/SKILL.md` 定义标准化的 `enhance_prompt` 接口，为 Claude Code 提供自动触发逻辑。
- **跨语言通信桥接**：采用 Python (LLM/服务层) 与 Node.js (UI 自动化层) 双栈架构，各 UI 代理 (`qoder_ui`, `codex_ui`) 通过 Chrome DevTools 协议捕获输入后，经由本地环回接口 (`localhost:8765`) 桥接至核心服务。
- **统一运维入口**：根目录 `package.json` 聚合了所有子模块的启停脚本、守护进程注册及 LaunchAgent 安装命令，屏蔽了底层多语言环境的配置差异。
