- **入口与生命周期**：`App.swift` 定义 `@main` 入口，配置 `.accessory` 激活策略以隐藏 Dock 图标，并通过 `AppDelegate` 管理窗口控制器与全局快捷键。
- **状态管理**：采用 `AppState` (ObservableObject) 作为单一数据源，通过 `@Published` 驱动 UI 响应，协调窗口几何变换、会话加载及增强任务状态。
- **UI 渲染**：`IslandView.swift` 基于 SwiftUI 构建，利用 `NotchInfo` 动态计算刘海物理尺寸，实现胶囊条与展开卡片的无缝过渡及安全区规避。
- **领域逻辑分层**：`SessionReader.swift` 负责解析本地 Claude Code 会话文件；`EnhanceClient.swift` 封装远程增强 API 调用；`Selection.swift` 处理剪贴板读写及底层按键模拟。
- **窗口控制**：自定义 `IslandWindow` 突破系统安全区限制，配合 `NoInsetHostingView` 确保内容在刘海区域正确绘制。
