- **入口层**：`server.py` 实现基于标准输入输出的 MCP 协议处理器（JSON-RPC 2.0），负责 `initialize`、`tools/list` 和 `tools/call` 等核心交互；`http_server.py` 提供本地 HTTP 接口（默认端口 8765），支持跨域请求以适配前端插件。
- **业务逻辑层**：`enhance.py` 封装核心增强逻辑，通过 Dashscope API（兼容 OpenAI 格式）调用大模型进行提示词重写，并包含简单的回退机制。
- **数据处理层**：`context_packaging.py` 定义 `PromptContext` 数据模型，负责对话历史、代码事实、项目摘要等上下文的组装、去重与智能截断，确保在有限的 Token 预算内传递高价值信息。
- **依赖方向**：入口模块依赖业务逻辑与数据处理模块，整体采用分层架构，各模块间通过函数调用解耦。
