- 以 `mcp-server` 为核心服务节点，对外暴露标准化的 `enhance_prompt` 工具接口，并通过 `skill/SKILL.md` 定义在 Claude Code 中的自动触发逻辑。
- `qoder-ui` 与 `codex-ui` 作为独立守护进程，利用 Chrome DevTools 协议注入前端脚本，捕获用户输入后通过本地 HTTP API (`localhost:8765`) 桥接至核心增强服务，实现跨应用的无缝体验。
- 根目录 `package.json` 提供统一的 Node.js 侧运维入口（如守护进程启停、LaunchAgent 安装），而 Python 侧服务则通过环境配置（如 `DASHSCOPE_API_KEY`）实现模型能力的动态加载。
